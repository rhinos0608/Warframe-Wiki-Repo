# Gravimag

Allows archwing guns to be deployed in terrestrial zones. Install this device on archwing guns to compensate for atmospheric and gravitational forces not experienced in space.
NOTE: Deployment requires gear obtained from the Profit-Taker mission in Fortuna.

## Overview

**Gravimag** is a rifle.
